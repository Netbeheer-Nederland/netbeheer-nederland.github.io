# Modellen van Netbeheer Nederland

Deze modellen vormen het fundament voor goede samenwerking binnen de sector én met externe partijen. Samen beschrijven ze een logisch geheel: van bedrijfscontext en betekenis tot brondata en het gebruik daarvan.

1. [**NBility**](https://nbility.netbeheernederland.nl)
  <br>Ordent wat een netbeheerder doet in samenhangende onderdelen en biedt zo de context voor de andere modellen.
    
2. [**Begrippen**](https://begrippen.netbeheernederland.nl)
  <br>Zorgt voor een gedeelde betekenis van begrippen tussen modellen.
    
3. [**Energieregisters**](/stelsel)
  <br>Beschrijft de authentieke brondata waarop de sector vertrouwt.
    
4. [**Dataproducten**](/data-products)
  <br>Laat zien hoe deze brondata worden gebruikt in gegevensuitwisseling.
